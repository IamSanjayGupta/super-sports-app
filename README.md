## For Developers

### Prerequisites

- Node.js v18 (recommended)
- npm or yarn
- Expo CLI

1. Install Expo CLI globally

   ```bash
   npm install -g expo
   ```

2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
   npx expo start
   ```
